# 🌌 플랫폼 THERE 소개

THERE는 상상력에 한계가 없는 우주공간을 컨셉으로 만들어졌습니다

다양한 컨셉으로 기획 / 디자인 / 컨텐츠를 구성하여 나만의 메타버스 커뮤니티를 5분만에 만들 수 있습니다
